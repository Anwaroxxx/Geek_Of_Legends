# Geek of Legends: Eclipse Raid

A dark fantasy 2D RPG boss-rush game built with React 19, Vite, and PixiJS.

## Features
- **Epic Boss Battles**: Face off against Chronos, Sauron, and Lilith.
- **Strategic Combat**: Turn-based mechanics with unique hero classes (Warrior, Mage, Archer).
- **Riddle System**: Test your logic and knowledge to gain the upper hand.
- **Rich Visuals**: Dynamic animations, VFX, and a premium dark aesthetic.
